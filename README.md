Requirements :

2- install matplotlib library

Installation:
1-clone the repository.
2- install required packages(pip install -r requirements.txt)

then generate sample data by run the file create_sample_data.py this step will generate: reservations.csv and routes.json

finally you can run the application by excute python main.py

The output generated by the program will include:

Loading of bus routes from the file routes.json
Loading of reservations from the file reservations.csv
Available seats in each bus route
Test reservation
Visualization of:
Occupancy bar chart
Reservation distribution pie chart
